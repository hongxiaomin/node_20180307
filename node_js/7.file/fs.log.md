# 路径处理 path
- join将多个参数值字符串结合为一个路径字符串
- resolve 取得绝对路径
- basename 获取一个路径中的文件名
- extname 获取一个路径中的扩展名
- path.sep操作系统指定的文件分隔符
- path.delimiter 属性值为系统指定的环境变量 路径分隔符
